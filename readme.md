# array-equal

> Check if two arrays are equal

It checks that the elements and order are the same.

## Install

```sh
npm install array-equal
```

## Usage

```js
import arrayEqual from 'array-equal';

arrayEqual([1, 2, 3], [1, 2, 3]);
//=> true

arrayEqual([1, 2, 3], [1, 2, 3, 4]);
//=> false
```
